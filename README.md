# Nest-Next Web Crawler

* node-json-db 연동
* HTML DOM 파싱(cheerio)

## Installation

```bash
$ npm install

$ npm run build
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
